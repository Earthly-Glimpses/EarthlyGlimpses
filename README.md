# 🚀 Website Update v1.0.1

**Pre-Release Date:** August 8, 2025

We're excited to share a new version of the site packed with visual improvements, smoother interactions, and better content organization.

---

## ✨ New Features

### 🔄 Preloader Animations
- Added smooth and lightweight **preloader animations**.
- Enhances user experience during initial page loads, especially for media-heavy pages.

### 🛠️ UI & UX Enhancements
- Refined layout with cleaner structure and improved responsiveness.
- Upgraded interactive elements and hover states.
- Better consistency across devices and resolutions.

### 🖼️ Improved Image Categories & Sorting
- Overhauled the **image categorization system** for more intuitive browsing.
- Implemented **dynamic sorting options** (by category, date, popularity).
- Optimized image loading and grid layout for faster rendering.

---

## 🐛 Bug Fixes
- Fixed layout issues on smaller screens.
- Resolved bugs related to image loading and broken links.
- Improved accessibility labels for screen readers.

---

## 🔧 Technical Changes
- Updated CSS and JavaScript for better maintainability.
- Codebase refactored for scalability and performance.
- Improved caching and image loading strategies.
